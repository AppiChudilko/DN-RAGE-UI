#### CircleMenu

Просто небольшое пояснение

`Параметры в state`

```
    showMenu — состояние отображения колеса
    selectedType — выбранный тип (
        0 - оружие
        1 - предметы
        2 - анимации
    )
    selected — индекс выбранной ячейки в колесе
    menuData — массив данных отображающихся в колесе
```

`Состав menuData`

```js
    menuData: [
        [
            {
              selected: number, // выбранный индекс из массива data
              circleInfoTop: string, // категория предмета, отображающиеся сверху
              data: [ // массив с предметами
                {
                  itemId: string, // ID предмета
                  ammount: string, // количество, пишется под картинкой предмета
                  circleInfoMiddle: string, // информация внутри колеса по центру
                  circleInfoBottom: string, // информация внутри колеса снизу
                }
              ]
            }
        ], // массив с оружием
        [...], // массив с предметами
        [...] // массив с анимациями
    ]
```
